Acumulus events
===============

The Acumulus component defines 3 events:

1) onAcumulusInvoiceCreated
2) onAcumulusInvoiceSendBefore
3) onAcumulusInvoiceSendAfter

See the example plugin AcumulusCustomiseInvoice for usage and documentation.
You can download this example extension from the Acumulus forum at
https://forum.acumulus.nl/index.php/topic,4642.0.html.
